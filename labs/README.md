## HR Analytics: Job Change of Data Scientists

### Контекст и Описание
Компания, работающая в сфере Big Data и Data Science, проводит курсы подготовки data scientists и хочет определить, какие кандидаты действительно планируют работать в компании после обучения. Это поможет оптимизировать затраты на обучение и улучшить планирование курсов.

Особенности данных:
- Несбалансированный датасет
- Преобладают категориальные признаки (номинальные, порядковые, бинарные)
- Некоторые признаки имеют высокую кардинальность
- Присутствуют пропущенные значения

### Целевая переменная
`target` - бинарный показатель:
- 0 - Не ищет смену работы
- 1 - Ищет смену работы

### Признаки
- `enrollee_id` - Уникальный ID кандидата
- `city` - Код города
- `city_development_index` - Индекс развития города
- `gender` - Пол кандидата
- `relevent_experience` - Релевантный опыт
- `enrolled_university` - Тип университетского курса
- `education_level` - Уровень образования
- `major_discipline` - Основная дисциплина
- `experience` - Общий опыт в годах
- `company_size` - Размер компании работодателя
- `company_type` - Тип работодателя
- `last_new_job` - Разница в годах с предыдущей работой
- `training_hours` - Пройденные часы обучения

### Задачи
1. Предсказать вероятность смены работы кандидатом
2. Интерпретировать влияние факторов на решение сотрудника

##  Metro Interstate Traffic Volume

### Контекст и Описание
Данные содержат почасовой объем трафика на запаbound I-94 между Миннеаполисом и Сент-Полом (Миннесота) с 2012 по 2018 год. Включает погодные условия и информацию о праздниках для анализа влияния на транспортный поток.

Характеристики:
- Многомерные, последовательные, временные ряды
- Типы признаков: целочисленные, вещественные
- Период: 2012-2018 гг.
- 48,204 наблюдений, 8 признаков

### Признаки
- `holiday` - Праздничные дни
- `temp` - Температура (в Кельвинах)
- `rain_1h` - Количество дождя за час (мм)
- `snow_1h` - Количество снега за час (мм)
- `clouds_all` - Облачный покров (%)
- `weather_main` - Основное описание погоды
- `weather_description` - Детальное описание погоды
- `date_time` - Дата и время измерения
- `traffic_volume` - Объем трафика (целевая переменная)

### Особенности данных
- Временные ряды с почасовой гранулярностью
- Включает погодные условия и праздники
- Подходит для задач регрессии и анализа временных рядов

### Задачи
1. Прогнозирование объема трафика
2. Анализ влияния погодных условий и праздников на загруженность дорог
3. Выявление сезонных и временных паттернов
